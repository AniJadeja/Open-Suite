# Media Directory

This directory is mapped to `/media` inside the Authentik server and worker containers.

## Purpose
Persists user-uploaded content and application assets.

## Contents
- **User Avatars:** Profile pictures uploaded by users.
- **Application Icons:** Logos and icons for connected applications.
- **Backgrounds:** Custom background images for flows.

## Important
**Do not delete this folder.** Doing so will result in the loss of all uploaded user content and application icons.
